游戏网关服务器GatewayServer的作用

  • 2011-11-21
  • mikzzz

GatewayServer
　　也称之为连接服务器，网络游戏的客户端一般是连接到这里，然后再由该连接服务器根据不同的需要，把游戏消息转发给其它相应的服务器（逻辑和地图服务器）。也因为它是客户端直接连接的对象，它同时也承担了验证客户身份的工作。

游戏网关服务器GatewayServer的作用：
游戏网关服务器可以作为客户端与 game server 的隔离作用
消息解析

与客户端保持连接，作为广播作用
消息合法性验证
转发消息到业务服务，针对不同的客户端消息分发到相应的服务处理
流量限制，消息分流作用
版本验证等
可扩展性、动态拓展
